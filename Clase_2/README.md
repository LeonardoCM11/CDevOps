# Formulario de datos personales 

# Datos Personales en Python

Programa básico que solicita datos del usuario (nombre, carrera, edad, peso y preferencia sobre fiesta) y los muestra en pantalla.

# Nota
El script pide al usuario:

Nombre completo

Carrera

Edad

Peso

Si le gusta la fiesta (True o False)

Luego imprime un mensaje con todos los datos ingresados
